DEPLOYMENT DOCUMENTATION

De stappen die werden ondernomen op de applicatie te maken op vscode via onze repo
https://github.com/EHB-MCT/full-projects-3-spetter#readme zijn als volgt:

1. Webpack installeren:
Eerst hebben we webpack geïnstalleerd door de startup guide te volgen via deze link:
https://webpack.js.org/guides/getting-started/
2. Tailwind installeren:
Nadat we webpack hebben geinstalleerd, hebben we de npm package Tailwind
geïnstalleerd via deze guide https://tailwindcss.com/docs/installation. En om van start te
gaan met deze package, hebben we een online video bekeken waar ze de startup van
TailwindCSS uitleggen via deze link https://www.youtube.com/watch?v=UBOj6rqRUME.
3. Tailwind CSS bootstrap:
Om het coderen te vergemakkelijken heeft Tailwind ook al een paar goeie bootstrap CSS
code klaar liggen voor ons. Zoals de pagina waarin alle verhalen staan doen we via
https://v1.tailwindcss.com/components/cards. En de code voor de tijdslijn doen we via
deze link waarin alle code al instaat
https://tailwindcomponents.com/component/simple-vertical-timeline
