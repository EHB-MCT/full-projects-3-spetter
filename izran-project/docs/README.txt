DEPLOYMENT DOCUMENTATION

De stappen die werden ondernomen op de applicatie te maken op vscode via onze repo https://github.com/EHB-MCT/full-projects-3-spetter#readme zijn als volgt:

De app is het best te zien op het formaat van de Iphone 12 pro.

1.	Webpack installeren:
Eerst hebben we Webpack geïnstalleerd door de startup guide te volgen via deze link:
https://webpack.js.org/guides/getting-started/
2.	Tailwind installeren:
Nadat we Webpack hebben geïnstalleerd, hebben we de NPM-package Tailwind geïnstalleerd via deze guide https://tailwindcss.com/docs/installation. En om van start te gaan met deze package, hebben we een onlinevideo bekeken waar ze de startup van TailwindCSS uitleggen via deze link: 
https://www.youtube.com/watch?v=UBOj6rqRUME
3.	Tailwind CSS-bootstrap:
Om het coderen te vergemakkelijken heeft Tailwind ook al een paar goeie bootstrap CSS-code klaarliggen voor ons. Zoals de pagina waarin alle verhalen staan doen we via
https://v1.tailwindcss.com/components/cards
En de code voor de tijdslijn doen we via deze link waarin alle code al instaat
https://tailwindcomponents.com/component/simple-vertical-timeline


Om de app te laten werken moet je volgende commands gebruiken in de terminal:
1.	cd .\izran-project\
2.	npm run build
3.	npm start
